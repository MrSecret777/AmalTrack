# Amal Track

Amal Track ialah aplikasi checklist amalan harian dengan dashboard harian, mingguan, dan bulanan.

## Fail

- `index.html` ialah fail tunggal yang sudah mengandungi HTML, CSS, dan JavaScript aplikasi.
- Tabler UI dipanggil melalui CDN.

## Cara guna di GitHub Pages

1. Letakkan `index.html` di root repository.
2. Aktifkan GitHub Pages daripada tetapan repository.
3. Pilih branch yang mengandungi fail ini.

Data checklist disimpan dalam `localStorage` browser pengguna.
